\section{WS 2.3 - 4 - Würfeln - ZO - BIFIE}

\begin{beispiel}[WS 2.3]{1}
Ein idealer sechsseitiger Würfel mit den Augenzahlen 1 bis 6 wird einmal geworfen.

Ordne den Fragestellungen in der linken Spalte die passenden Wahrscheinlichkeiten in der rechten Spalte zu.

\zuordnen[0.35]{
				R1={Wie groß ist die Wahrscheinlichkeit, dass
eine gerade Zahl gewürfelt wird?},				% Response 1
				R2={Wie groß ist die Wahrscheinlichkeit, dass
eine Zahl größer als 4 gewürfelt wird?},				% Response 2
				R3={Wie groß ist die Wahrscheinlichkeit, dass
eine Zahl kleiner als 2 gewürfelt wird?},				% Response 3
				R4={Wie groß ist die Wahrscheinlichkeit, dass
eine Zahl größer als 1 und kleiner als 6
gewürfelt wird?},				% Response 4
				%% Moegliche Zuordnungen: %%
				A={$\frac{1}{3}$}, 				%Moeglichkeit A  
				B={$\frac{1}{6}$}, 				%Moeglichkeit B  
				C={$\frac{1}{2}$}, 				%Moeglichkeit C  
				D={$1$}, 				%Moeglichkeit D  
				E={$\frac{5}{6}$}, 				%Moeglichkeit E  
				F={$\frac{2}{3}$}, 				%Moeglichkeit F  
				%% LOESUNG: %%
				A1={C},				% 1. richtige Zuordnung
				A2={A},				% 2. richtige Zuordnung
				A3={B},				% 3. richtige Zuordnung
				A4={F},				% 4. richtige Zuordnung
				}
\end{beispiel}
