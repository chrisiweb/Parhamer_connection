\begin{beispiel}[WS 2.4]{1}
Gegeben sind Ausdrücke mit Binominialkoeffizient. 

Ordne den vier Ausdrücken jeweils das passende Ergebnis (aus A bis F) zu!

\zuordnen[0.2]{
				R1={$\binom{4}{0}+\binom{4}{1}+\binom{4}{2}+\binom{4}{3}+\binom{4}{4}=$},				% Response 1
				R2={$\binom{15}{3}+\binom{15}{4}=$},				% Response 2
				R3={$\binom{16}{9}=$},				% Response 3
				R4={$\binom{15}{15}=$},				% Response 4
				%% Moegliche Zuordnungen: %%
				A={$\binom{16}{1}$}, 				%Moeglichkeit A  
				B={$\binom{16}{4}$}, 				%Moeglichkeit B  
				C={$\binom{16}{7}$}, 				%Moeglichkeit C  
				D={$\binom{16}{16}$}, 				%Moeglichkeit D  
				E={$\binom{15}{7}$}, 				%Moeglichkeit E  
				F={$\binom{15}{1}$}, 				%Moeglichkeit F  
				%% LOESUNG: %%
				A1={A},				% 1. richtige Zuordnung
				A2={B},				% 2. richtige Zuordnung
				A3={C},				% 3. richtige Zuordnung
				A4={D},				% 4. richtige Zuordnung
				}
